# Structure Area Selector

A client-side Fabric mod that makes saving structures way less painful. Instead of messing around with the structure block UI to set offsets and sizes manually, you just click the two corners of your build in-world and it handles the rest.

## How to use

1. Place a structure block in creative mode — a config screen will pop up
2. Type your structure name (supports `modid:name` format like `mvs:my_house`) and hit Begin Selection
3. Left click to set the first corner
4. Right click to set the second corner
5. Press G to save (or Enter)

Thats basically it. You can also scroll to nudge a corner along the direction you're looking, and middle click to expand the selection to include a block you missed. The box turns red if you go over the 48 block limit. Press X or Escape to cancel at any time.

Keybinds are fully configurable through the vanilla controls menu.
